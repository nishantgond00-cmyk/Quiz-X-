# Quiz X

Quiz X is an offline progressive web app (PWA) quiz application.

## Features
- 1000+ questions (GK, GS, Physics, Mathematics)
- Works offline after first load
- App-like experience on mobile
- Clean and simple UI

## How to use
1. Open the app link in Chrome
2. Click on "Add to Home Screen"
3. Use it like a real mobile app
